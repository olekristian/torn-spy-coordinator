How To Use This Page
  1. Enter your display name at the top of the page.
  2. Employees must contact Kattemannen [2670127] to get the access code.
  3. Enter the access code and press Save, then press Refresh.
  4. Go to Claim Board and press Claim on the target you are working on.
  5. Paste the spy result into the target card and press Submit Result.
  6. If something looks wrong, press Refresh and check the message shown next to the top buttons.

Setup
  1. Create a Google Sheet tab named Tasks with headers (row 1):
    id | createdAt | status | targetName | targetId | notes | claimedBy | claimedAt | submittedAt | payloadJson | formatted
  2. Open Extensions → Apps Script. Paste the backend code I provided into Code.gs. Deploy: Deploy → New deployment → Web app. Copy the URL.
  3. Paste that URL into API Base URL above. (Optional) set a Script property API_KEY and paste the same value here.
